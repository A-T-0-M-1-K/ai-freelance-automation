# Changelog

## [0.1.0] - 2024-01-10
### Добавлено
- Базовая архитектура проекта с модулями `ai_core` и `app`
- Конфигурационная система с JSON схемами
- Модели данных для заказов, клиентов, сообщений
- GUI интерфейс на PyQt5 (базовый каркас)
- Документация и скрипты установки

### Исправлено
- Циклические импорты между модулями
- Конфликты зависимостей в pyproject.toml
- Отсутствующая точка входа приложения

### Известные проблемы
- Нет работающей интеграции с фриланс-платформами
- AI модели не загружены и не настроены
- База данных требует миграций

- Создана базовая структура проекта
